Snowem

is a lightweight live streaming server, based on webrtc technology. Its design mainly focuses on simplicity, scalability and high performance.

Check Demo View on GitHub

Designed for developers

Simple API allows developers easily integrate video streams while staying focus on logic of their applications.

Cross platforms

Webrtc technology is available on most platforms:

  • Web browers: Chrome, Opera and Firefox.
  • Mobile platforms: Android and iOS.

High Performance

Written in C with focus on performance, Snowem provides excellent real-time communications.

Get ready for live streaming in your application

Snippet of javascript integration

window.snowAsyncInit = function() {
   SnowSDK.init(function (){ 
      var config = { 'servername': "snowem.example.com" };  
      // Create peer object
      var peer = SnowSDK.createPeer(config);
      peer.createChannel({name: "demo"},function (peer) {
         var settings = { 
            'channelid': peer.channelId,
         };
         // Publish a stream with a channel.
         peer.publish(settings);
      }); 
   }
}
